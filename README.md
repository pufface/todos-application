# Todos Web App

## Prerequisits
- JRE 8
- Node 6.10
- Npm 4.3.0

## Todos Database
run database: `$gradlew run`

## Todos Service
run server: `$gradlew bootRun`
create fat jar: `$gradlew bootRepackage`
create eclipse project: `$gradlew eclipse`

## Todos Frontend
install: `npm install`
serve app: `npm run serve`
run app dev: `npm start`
to more see package.json
